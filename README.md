# FUTURE_DS_03

About the Task
In this task, you will analyze marketing and lead funnel data to understand how users move from visitors → leads → customers.

This type of analysis is core work in startups, digital marketing agencies, SaaS companies, and sales teams, where improving conversion rates directly increases revenue.

Your goal is to help a business answer questions like:

Where are users dropping off in the funnel?
Which channels bring high-quality leads?
How can conversion rates be improved?
Which stages need optimization?
